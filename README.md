## Overview
MADE-T is a probabilistic shear-strength estimator for high-strength steel (HSS) reinforced concrete columns. It combines a Feature-Tokenizer Transformer ensemble, mechanics-informed engineered features, measurement-uncertainty data augmentation, transfer learning from a conventional-RC column database, and split-conformal prediction intervals. The estimator delivers a point prediction together with a calibrated 90% prediction interval and per-sample aleatoric and epistemic uncertainty.

An interactive browser demonstration is available at the <a href="https://twakjira.github.io/MADE-T-HSS/" target="_blank" rel="noopener noreferrer">project page</a>.
